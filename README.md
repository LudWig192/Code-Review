# LAB3 – Simulação de Fluxo Profissional com Code Review
- 📚 Disciplina: DevOps
- 🧪 Atividade: LAB3
------------------------------------
## 📌 Sobre o Projeto
Este projeto foi desenvolvido como parte da atividade LAB3 da disciplina de DevOps, com o objetivo de simular um fluxo profissional de desenvolvimento de software utilizando boas práticas aplicadas no mercado.
A proposta foi reproduzir o funcionamento de uma equipe real, utilizando:
- Git para controle de versão
- GitHub para colaboração
- Branches organizadas
- Pull Requests
- Code Review
- Regras de proteção de branch 

O foco principal foi garantir que apenas código revisado e aprovado chegasse à branch principal do projeto.
